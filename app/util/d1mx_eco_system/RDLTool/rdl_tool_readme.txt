1. Introduction

   This program will generate a DL(DisplayList) to be used in VOWE. Please use as a technical
  reference to software development when your DL for this program, and generated by using the
  present program.
   
   
  ****************************** Caution *********************************
   This program is for reference all, it does not guarantee its operation.
  The thorough assessment of the entire system If you want to include in
  the actual system, please judge the applicability of the information at
  your own risk.
  ****************************** Caution *********************************

2. File structure

  I shows the file structure of the DL generation program below.
  
  rdl_tool_readme.txt              : This file.
  Launcher.exe                     : DL generation program body.(User will run this exe file)
  filters                          : DL generation filter program
   +-- fisheye_lite.exe
   +-- FanwiseDL.exe
   +-- TextToDL.exe
   +-- readme.txt
  operationTools                   : Various tools such as VOWE emulator.
   +-- DLGenerator.exe
   +-- Inter-processingCommunication.dll
   +-- vowe_render.exe
   +-- voweemu.dll
   +-- readme.txt
   
3. DL generation procedure
  See DL_Tool_manual_Rev0.3_20140925.pdf

4. RDL tool change history

 Ver.1.7.1.1:
 
   When InputSize is set to (0,0), the image does not appear when you
  change to ColorBar or CheckerBoard the FilterType. Therefore, I have
  to be changed to (width = 320, height = 240) InputSize of default.

 Ver.1.7.1.0:
 
   Add a InputImage column to Common configuration, it was able to be 
  selected from the following three types: the type of InputImage. 

     1. file specification
     2. checker board 
     3. color bar

 Ver.1.7.0.0:
 
    Added the function to change Input Image File by dragging and dropping
   the image file to Input Image window. 

    "Open File" in the dialog to specify the InputImage, Change image files
   so that it is only displayed. 

    Added the function to output as an image file an image being displayed
   in the current Output Image window. 
    -> to output a file of Output Image window on the "Output to Image file"
      button pressed (can output format: *.BMP, *.JPG) 
      
    DL file generation, not just the *.h files, Modified to be able to
   output the combination *.c + *.h file.
    -> specify the menu "Setting (s)" -> "Separate DL File (S)"

5. Notes
   If the image after the correction does not appear in the OutputImage window,
  you may want to improve by setting again the source image file in SourceFileName.
